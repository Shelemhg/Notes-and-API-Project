VARIABLES

Scope

Const and let   only exist inside the block they are declared

 

Any variable outside a block is global

 

Variables are case sensitive

 

\ skips a special character  I.E. 'It\'s me'

 

 

 

DOT NOTATION

You can access properties of strings with a dot IE 

 

name.length; // retrieve the name variable's length property

<< 5

 

METHODS

name.toUpperCase();

<< 'ALEXA'

 

'Java' + 'Script' + ' ' + 'Ninja';

<< 'JavaScript Ninja'

 

 

INTERPOLATION

 

const name = `Siri`;

`Hello ${ name }!`;

<< 'Hello Siri!'

 

const age = 39;

`I will be ${ age + 1 } next year`;

<< 'I will be 40 next year'

 

 

NUMBER METHODS

5..toExponential();

>> "5e+0"

 

 

 

ARRAYS

avengers = [ ...avengers, ...['Hulk','Hawkeye', 'Black Widow'] ];

<< ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

 

 

 

 

 

FUNCTION

 

DEFAULT VALUE FOR PARAMETER

function hello(name='World') {

    console.log(`Hello ${name}!);

}

 

 

Default parameters should always come after non-default parameters, otherwise default values will always have to be entered anyway.

 

 

ARROW FUNCTIONS

 

Const NAME = PARAMETERS => MAIN THING;

 

const add = (x,y) => x + y;
